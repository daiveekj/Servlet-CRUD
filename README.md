A simple **CRUD web application** using **Jakarta Servlets (Tomcat 10)**, **JSP**, and **MySQL**.  
This project demonstrates **Create, Read, Update, Delete** operations for a `users` table.
 Technologies Used

- Java 17
- Jakarta Servlet 5 / JSP 3 (Tomcat 10)
- JSTL 2.0
- MySQL 8.x
- Maven 3.x
- Apache Tomcat 10.x
